# Jellysmack - Technical Test

A web application based on [Rick and Morty API](https://rickandmortyapi.com/documentation/).

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Run your unit tests
```
yarn test:unit
```

### Lints and fixes files
```
yarn lint
```
